# EcoVida 🌿 - Pureza en Sostenibilidad

**EcoVida** es una plataforma web moderna diseñada para promover servicios de asesoría ambiental y la comercialización de productos orgánicos. Este proyecto fue desarrollado como parte de la formación en **Talento Tech 2026**, enfocándose en la creación de interfaces de usuario (UI) atractivas, limpias y altamente funcionales.

## 🚀 Características del Proyecto

* **Interfaz Consciente:** Diseño basado en una paleta de colores tierra y bosque, alineada con la identidad de marca sostenible.
* **Componentes Reutilizables:** Implementación de tarjetas de testimonios y servicios optimizadas mediante Flexbox para garantizar simetría visual.
* **Efectos Visuales Avanzados:** * *Glassmorphism* en el encabezado mediante `backdrop-filter`.
    * Efectos de profundidad con `box-shadow` personalizados.
    * Animaciones suaves de zoom en la galería de productos.
* **Formulario de Contacto:** Estilizado con mejoras en la experiencia de usuario (UX) como paddings internos y placeholders personalizados.

## 🛠️ Tecnologías y Metodologías

* **HTML5 Semántico:** Estructuración del contenido para accesibilidad y SEO.
* **CSS3 Moderno:** Uso intensivo de **Flexbox** para layouts dinámicos y alineación de contenido.
* **SVG:** Gráficos vectoriales para iconos y elementos decorativos que mantienen la nitidez.
* **IA Generativa:** Uso estratégico de herramientas de IA para la creación de imágenes de testimonios y prototipado de recursos visuales.

## 📁 Estructura del Repositorio

* `/img`: Directorio con todos los recursos visuales, iconos y fondos.
* `/css`: Hojas de estilo organizadas por secciones.
* `index.html`: Archivo principal con la estructura del sitio.
